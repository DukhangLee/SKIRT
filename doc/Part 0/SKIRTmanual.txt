\page SKIRTmanualindex SKIRT manual

This manual is structured as follows:

<b> Part I: Guides -- provides overall guidance and tutorials. </b>

  - The \ref InstallationGuide describes how to install SKIRT and its dependencies on your computer.
  - The \ref UsersGuide explains how to run a SKIRT simulation and what output to expect.
  - The \ref FSUsersGuide explains how to prepare for and run a FitSKIRT simulation and what output to expect.
  - The \ref DevelopersGuide offers guidelines on how to extend or adjust the SKIRT code.

<b> Part II: Concepts -- offers extra information on selected topics. </b>
    
  - \ref DustProp describes how SKIRT deals with optical and calorimetric dust properties.
  - \ref PerInstr provides details on SKIRT's perspective instrument, used mostly for making movies.
  - \ref Cfitsio introduces the library included with SKIRT to read and write FITS files.
  - \ref voro introduces the library included with SKIRT to construct Voronoi tesselations.
  - \ref GAlib introduces the library included with FitSKIRT to search the model parameter space.

<b> Part III: Reference -- documents every class, namespace and function. </b>

If you are browsing an HTML version of this documentation:

  - select the "Related Pages" tab at the top of the help window to access the information in Part I and II;
  - select the "Classes" tab at the top of the help window to access the reference material.
